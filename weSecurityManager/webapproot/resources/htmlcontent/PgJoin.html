PgJoin Help

This page will assist you in registering this application to the current Wave ERP System.

Showing / Editing Database Connection Properties

1. Enter the [Updating Password] and click [Show Property] button.
2. Fill up the [Username], [Connection URL] and [Password]
3. Click the [Update Database Connection] button to update the connection profile.
4. To generate rights to Admin, set the target [Entity] click [Join This App to Wave ERP.] button.


To connect to the ERP application, URL must be allowed first. If a URL is not registered, the system won't allow a connection. The process below will register the [Local host] as an allowed URL.

Allowing Local Connection

1. Enter in the license key in the [Registration Key]
2.  Click the [Register] button.

Connection to all apps is not possible using localhost.
