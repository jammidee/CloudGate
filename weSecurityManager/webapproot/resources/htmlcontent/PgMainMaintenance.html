PgMainMaintenance Help

This page list all the available maintenance pages.
  • Entity - display all the available Entity in the system.
  • App List - list all the available registered apps in the Wave ERP system.
  • Logs - display all the logs generated by all the applications registered to Wave ERP.
  • Regs List - display all the available registry information of the entire Wave ERP.
  • Table List - list all the available tables in Wave ERP.
  • Stats - display graphs related to the entire Wave ERP.
